Primeiros Computadores

Voltar para o inicio

Também conhecido como Electronic Numerical Integrator And Computer,
Eniac foi mostrado ao mundo em 14 de fevereiro de 1946.

Foi desenvolvido para a 2° Guerra Mundial, esse enorme computador tinha nada mais que 30 toneladas
são mais ou menos 7 elefantes e ocupava uma 180 m² de área, contava com um hardware de 70 mil resístores e 18 mil válvulas
consumiam 200mil watts de energia, um verdadeiro Gigante!!

Ao sair de cena o ENIAC foi desmontado e diversas de suas peças estão em exposição pelo mundo.
Os computadores ainda ficaram exclusivamente nas mãos de empresas e órgãos governamentais até o final da década de 70,
quando a computação doméstica começou a dar seus primeiros passos.
Mas isso é história para um outro Especial.

Html é ou não uma linguagem de Programação?

Voltar para o inicio

O que significa Html? em inglês Hyper Text Markup, linguagem de marcação de hipertexto
o Html não é uma linguagem de Programação e sim de marcação de textos através das Tags Html está um pouco longe de ser uma linguagem de programação,
Html não toma decisões, não faz calculos,
não cria rotinas ou funções.

O Html tem códigos para criação de páginas Web, através das Tags, essas tags definem tamanho de fonte,
cor espaçamento e vários outros aspectos de um site
Html podem ser usados para criar as estruturas dos sites parágrafos, links e alguns elementos usados, tags.
Html é considerado padrão da Internet pelo W3C
O Html é o corpo de um site, o Css é o que deixa o site visivelmente mais estilo e bonito.
